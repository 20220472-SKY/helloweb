HTML 기본 구조

<!DOCTYPE html>HTML5 문서 선언
<html>HTML 문서 전체
<head>문서 정보 영역
<body>화면에 표시되는 내용

Head tag

<title>웹페이지 제목
<meta>문서 정보 설정
<link>CSS 연결
<script>JavaScript 연결
<style>CSS 직접 작성

글자 태그

<h1>가장 큰 제목
<h2>두 번째 제목
<h3>세 번째 제목
<h4>네 번째 제목
<h5>다섯 번째 제목
<h6>가장 작은 제목
<p>문단
<br>줄바꿈
<hr>구분선
<strong>굵게 강조
<em>기울임
<span>부분 스타일

목록 태그

<ul>순서 없는 리스트
<ol>순서 있는 리스트
<li>항목

링크 태그

<a>하이퍼링크
href주소 지정
target="_blank"새 창 열기

리스트 태그
